# MVP 최종 운동 세트 확정본 (20~22개)

본 세트는 **웹캠 안정성 + 임상 재활 타당성 + 구현 난이도**를 기준으로 선정한  
“최소하지만 확장 가능한” MVP 구성이다.

---

## A. 전신 운동 (14개)

### 하체 (6)
| ID | 한글명 |
|----|------|
| squat | 스쿼트 |
| wall_squat | 벽 스쿼트 |
| chair_stand | 의자 앉았다 일어나기 |
| straight_leg_raise | 누워서 다리 들기 |
| standing_march_slow | 서서 천천히 행진 |
| seated_knee_lift | 앉아서 무릎 들기 |

### 상체 (4)
| ID | 한글명 |
|----|------|
| standing_arm_raise_front | 팔 앞으로 들기 |
| shoulder_abduction | 어깨 벌리기 |
| elbow_flexion | 팔꿈치 굽히기 |
| wall_push | 벽 밀기 |

### 코어/체간 (4)
| ID | 한글명 |
|----|------|
| seated_core_hold | 앉아서 코어 버티기 |
| standing_anti_extension_hold | 서서 허리 버티기 |
| standing_arm_raise_core | 코어 유지하며 팔 들기 |
| bridge | 브릿지 |

## B. 손 재활 운동 (8개)

| ID | 한글명 |
|----|------|
| finger_flexion | 손가락 굽혔다 펴기 |
| finger_spread | 손가락 벌리기 |

## MVP 선정 이유 요약
- 웹캠 단일 시점에서 **인식 안정성 확보**
- 낙상/통증 위험이 낮은 동작 위주
- 재활 단계(급성→만성) 전부 커버 가능
- 이후 dead_bug, bird_dog 등 연구용 확장 용이
